An interactive **Web Analytics Dashboard** created using **Power BI** is included in this repository.  The dashboard facilitates the visualisation and analysis of a website's essential performance information, empowering stakeholders to make informed decisions.


 ## Features

 The percentage of visitors who depart after only viewing one page is shown by the "bounce rate."
 - **Conversion Insights**: Monitors crucial conversion metrics to assess target accomplishment.
 The "Traffic Funnel" illustrates the user's path across the various phases of the website.
 - **Monthly User Interaction**: Displays monthly patterns in user involvement and activity.
